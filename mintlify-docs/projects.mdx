# 项目经验

## 流水线物料组装搬运

**2025.12 - 2026.05 / VisionMaster、TCP、汇川机械臂、7 轴组装、MES 对接**

### 项目概述

基于皮带输送、视觉定位、机械臂抓取与转盘协同的自动化组装搬运项目。物料到位后由海康 VisionMaster 进行模板匹配定位，并通过 TCP 将坐标发送给汇川机械臂完成抓取、装配与搬运。

### 个人职责

独立负责自动化开发流程、架构规划和技术选型；完成上位机视觉模板匹配、机械臂取件逻辑和 7 轴组装控制开发；规划 PLC、视觉、机械臂与 MES 之间的通信方式，并完成本地数据与客户 MES 系统对接。

### 项目成果

实现工件识别、精准定位、自动抓取与组装联动，提升产线自动化程度与组装效率；完成产量数据联网和实时展示，支持远程查看生产状态与灵活排产。

## 五轴上下料

**2025.08 - 2025.09 / S7-1200、多轴协同、联锁控制、异常报警**

### 项目概述

上游皮带将工件输送至工作区，接近开关检测来料后，设备通过多轴协同完成取料与放料，并输送至下游工位。

### 个人职责

独立负责项目自动化开发流程与结构规划，基于西门子 S7-1200 完成五轴上下料设备控制程序开发，包含来料检测、动作联锁、节拍控制、异常报警与现场调试。

### 项目成果

完成五轴设备稳定控制与上下游衔接，提升取放料流程稳定性，并为后续功能扩展与项目复用打下基础。

## CCD 视觉检测

**2023.12 - 2025.06 / Flutter、OpenCV、Docker、Clonezilla、追溯数据**

### 项目概述

工人扫码录入钣金编号后放置到皮带，系统触发多相机拍照，对螺钉、螺帽、沉孔、丝印字符等进行模板匹配和检测。检测结果本地精简存储，并将完整数据上传工厂本地服务器，实现追溯查询。

### 个人职责

负责上位机开发流程规划，以及下位机、本地数据、远端服务器之间的数据流设计；使用 Flutter 开发上位机界面，集成 OpenCV 模板匹配算法并完成本地数据存储开发；使用 Docker 容器化打包部署，通过 Git 进行版本管理，并结合 Clonezilla 完成统一系统镜像部署与升级方案设计。

### 项目成果

降低混料、错料和漏检风险，提升出货质量稳定性；建立检测数据本地与服务器双层存储机制，实现产品数据可追溯、可查询；通过模块化部署与统一环境方案降低现场升级维护成本。

## 智能钥匙柜

**2022.03 - 2023.05 / Android、人脸识别、RFID、Java、MongoDB**

### 项目概述

面向车辆钥匙管理场景的智能设备项目。用户通过人脸识别查询当日工单并领取绑定钥匙，工单完成后归还钥匙，系统自动更新线上工单状态，实现派车、用车、还车全流程数字化管理。

### 个人职责

负责上位机功能规划、多版本兼容与客户数据平台对接统筹；使用 Android 开发设备端上位机程序，集成人脸识别、扫码枪和超高频 RFID 读取器；使用 Java + MongoDB 开发后端接口，并完成与客户数据平台的数据联动。

### 项目成果

实现车辆管理与业务平台绑定，打通派车、取钥匙、还钥匙和工单状态更新流程，推动客户车辆管理向智能化、数字化转型。

## 千里眼

**2020.04 - 2022.05 / Flutter、Android、iOS、Web、报表邮件**

### 项目概述

面向现场排工排产、异常上报与质量追踪的移动端与 Web 协同系统。现场人员可对异常工件进行按钮触发上报、图片拍照和问题描述提交，系统自动汇总生成不良报表并邮件发送给管理人员。

### 个人职责

参与移动端和 Web 端技术架构选型及开发规划；使用 Flutter 进行移动端开发，兼容 Android 与 iOS 并完成应用上架；针对应用稳定性、多平台兼容性和客户使用便捷性进行优化。

### 项目成果

帮助客户实现排工排产与异常数据上报数字化，提高现场质量监测效率和管理透明度。
